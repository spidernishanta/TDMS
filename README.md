# Telephone Directory Management System

  Telephone Directory Management System is an application aimed at storing the telephone numbers along with the details of the person concerned that will include address of a particular person. Developed as a learning project under C++ programming language , this application is built based on the concept of Data Files, Data Structures and Class / Structures. Using this application user can easily store the details of any person and based on some particular data values different types of information can be viewed by this application. 

  However care must be taken to ensure correct kind of data entered while using this application. This application can be made more attractive by using graphics which is beyond the scope of our knowledge. Your system must have support for Ms- DOS OPERATING system to run this application.

  The interactive menu will guide you for the detail operating process. Still if needed help option will be always with you to be referred which is also included in the program menu.
